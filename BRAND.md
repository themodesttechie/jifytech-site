# JifyTech — Brand Kit

Derived from the official JifyTech logo (Looka export, `/brand/src/Logo Files/`). Single source of truth for web + print.

## Logo
- **Mark:** scalloped navy badge, white wavy border, four hanging coral lightbulbs (one lit), serif "JifyTech" wordmark, "JUST IN FOR YOU" coral tagline.
- **Primary lockup:** `assets/logo-badge.svg` — full colour badge, use on light or navy backgrounds.
- **Reversed:** `assets/logo-white.svg` — white, for navy/photo backgrounds (used in site header + footer).
- **Mono:** `assets/logo-color.svg` (coral+navy, no background) for light backgrounds.
- Print masters (EPS/PDF, black/white/colour) in `brand/src/Logo Files/For Print/`.
- **Clear space:** keep padding ≥ 1× the badge border width on all sides. **Min size:** 32px (digital), 12mm (print). Never recolour, stretch, rotate, or add effects.

## Colour
| Role | Hex | Use |
|------|-----|-----|
| Navy (primary) | `#113a5d` | Backgrounds, headings, primary surfaces |
| Navy deep | `#0d2e49` | Footer, shadows, hover depth |
| Navy soft | `#1c4d77` | Secondary navy surfaces |
| Coral (accent) | `#ff7a8a` | CTAs, accents, eyebrows, icon strokes |
| Coral deep | `#e85d6e` | Coral on light bg (contrast), links |
| White | `#ffffff` | Reversed type, badge border |
| Cream | `#f6f4ef` | Alt section background |
| Ink | `#15212e` | Body text |
| Muted | `#5d6b7a` | Secondary text |

Coral on navy = brand signature. For body text on light, use Ink; for coral links on light, use Coral deep (AA contrast).

## Typography
- **Display / headings:** Playfair Display (700–800) — matches the serif wordmark.
- **Body / UI:** Inter (400–600).
- **Eyebrows / tagline:** Montserrat (600–700), uppercase, letter-spacing `.22em` — echoes "JUST IN FOR YOU".
- Loaded via Google Fonts in `index.html`.

## Voice
Tagline: **"Just in for you."** Responsive, on-demand, accountable. Plain-spoken, no hype. Lead with the customer's saved time, not the tech.
